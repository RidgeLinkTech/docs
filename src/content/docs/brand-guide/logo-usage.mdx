# Logo Usage

## Logo variants

- Horizontal logo
- Stacked logo
- Icon mark
- White logo
- Black logo
- Transparent PNG
- SVG
- Favicon
- Apple Touch Icon
- Android icons

## Rules

- Do not stretch the logo.
- Do not recolor outside approved palette.
- Do not place on visually noisy backgrounds.
- Prefer SVG for web and documentation use.

RidgeLink personas represent fictional employees used across demos, documentation, screenshots, and architecture scenarios.

- ServiceNow users and assignment groups
- Active Directory users
- Microsoft 365 users
- Entra ID users
- Approval workflows
- Incident response scenarios
- Architecture diagrams
- Blog authors
- Training examples

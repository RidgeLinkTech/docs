# Persona Image Standards

## Required images

- Headshot
- Full body image
- Office portrait
- LinkedIn-style image
- Casual professional image

## Requirements

- Consistent lighting
- Professional clothing
- Realistic facial features
- Neutral or RidgeLink-branded background
- Source files stored in the private `personas` repository

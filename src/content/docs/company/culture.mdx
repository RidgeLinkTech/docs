# Culture

## Communication

RidgeLink communication should be clear, direct, professional, respectful, and documentation-focused.

## Meeting Standards

- Meetings should have an agenda.
- Decisions should be documented.
- Action items should have owners.
- Follow-up should be tracked.

## Engineering Philosophy

1. Secure the system.
2. Make it reliable.
3. Automate the repeatable parts.
4. Document the design.
5. Improve continuously.

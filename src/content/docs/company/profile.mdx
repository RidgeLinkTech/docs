# Company Profile

**Company Name:** RidgeLink Technologies  
**Short Name:** RidgeLink Tech  
**Domain:** `ridgelink.us`  
**Headquarters:** Frederick, Maryland

RidgeLink Technologies is a fictional enterprise engineering lab created to demonstrate modern infrastructure engineering, cloud architecture, cybersecurity, automation, documentation, and software development practices through realistic business scenarios.

## Purpose

RidgeLink provides a consistent fictional company for GitHub portfolio repositories, documentation examples, architecture diagrams, ServiceNow demonstrations, Microsoft 365 labs, Active Directory labs, PowerShell automation examples, cloud architecture examples, cybersecurity scenarios, and homelab documentation.

## Positioning

RidgeLink should be presented as a fictional enterprise simulation, not as an incorporated company providing real commercial services.

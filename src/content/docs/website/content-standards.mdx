# Content Standards

## Page standards

Each website page should include a clear title, short summary, main call to action, internal links, consistent tone, and accessible headings.

## Writing principles

- Be clear.
- Avoid fluff.
- Use concrete examples.
- Keep service descriptions practical.
- Do not imply RidgeLink is a real incorporated company.

# Branding Prompts

## Logo prompt

```text
Create a modern technology company logo for RidgeLink Technologies.
Use a professional enterprise style with navy blue, medium blue, white, and light ice-blue accents.
Incorporate a subtle ridge or mountain motif and a connection/link concept.
Clean vector-style design, suitable for SVG recreation, GitHub avatar, favicon, website header, and documentation.
No mockups. No background texture. No tiny unreadable text.
```

## Favicon prompt

```text
Create a square favicon icon for RidgeLink Technologies.
Use a simplified RL monogram with a ridge or mountain shape and a subtle link motif.
Navy blue background, white and light blue foreground, high contrast, readable at small sizes.
Flat vector style, no text other than the monogram.
```

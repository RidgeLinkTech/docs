# Getting Started

Start with the foundation before creating additional repositories or assets.

## Initial setup

1. Create the GitHub organization.
2. Create the core repositories.
3. Deploy the documentation site.
4. Deploy the marketing website.
5. Populate the persona library.
6. Add branding and design assets.

## Core repositories

| Repository | Purpose | Visibility |
| --- | --- | --- |
| `.github` | Organization profile and community health files | Public |
| `website` | Public marketing website | Public |
| `docs` | Documentation hub | Public |
| `branding` | Brand assets and source design files | Private |
| `design-system` | Reusable components, styles, and templates | Private initially |
| `personas` | Persona profiles, metadata, and image sources | Private initially |

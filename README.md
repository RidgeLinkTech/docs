# RidgeLink Technologies Documentation

Documentation site for the fictional RidgeLink Technologies enterprise engineering environment.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Cloudflare Pages

- Framework preset: Astro
- Build command: `npm run build`
- Build output directory: `dist`
- Production branch: `main`
- Custom domain: `docs.ridgelink.us`
